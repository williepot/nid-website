Forest honey bee cream

"All-In-One" treatment for facial skin
Contains no harmful chemicals
Rich in vitamin C / AHA
Can be applied to all skintypes
Hypoallergenic
"Forest Honey Bee Cream" from B' secret brand is a 100% forest honey bee extract facial treatment. It contains 10 times more skincare properties than regular honey. This cream is an “all–in-one” treatment for facial skin. Forest Honey Bee Cream can be used with skincare cream. Importantly, it is free from hazardous chemicals (it is advisable to use only one cream at a time for better results). Sunscreen should be used after applying the cream regularly as Forest Honey Bee Cream is rich in vitamin C / AHA that help to exfoliate the skin, therefore the application of sunscreen can prevent exfoliated skin from direct sun exposure. Forest Honey Bee Cream can be applied to sensitive skin / skin with acne / skin affected by chemicals or darkened skin, melasma, freckles and dark spots. Those allergic to fruit acids / AHA can use Forest Honey Bee cream because a special anti-allergic substance for all skin types has been added.

Pregnant women and breastfeeding mothers should avoid using this product as it contains concentrated AHA.
